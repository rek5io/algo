## Wymagania

Aby uruchomić dany projekt, należy mieć zainstalowanego **Rusta**.

Instalacja Rust (rustup):
[https://www.rust-lang.org/tools/install](https://www.rust-lang.org/tools/install)

Sprawdzenie instalacji:

```bash
rustc --version
cargo --version
```

## Uruchomienie

```bash
cargo run --release
```
